# Sophat Odom Portfolio

A clean Vue 3 + Vite portfolio website for a Computer Science student applying for AI/software engineering opportunities.

## Run locally

```bash
npm install
npm run dev
```

## Build for deployment

```bash
npm run build
```

## Main files

- `src/data/portfolio.js` — personal content, projects, skills, education
- `src/components/` — reusable Vue sections
- `src/styles/main.css` — global design and responsive layout

## Deploy

This project can be deployed to Vercel or Netlify. For Vercel, import the folder/repository and use the default Vite settings.
